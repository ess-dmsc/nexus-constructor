=======
NXarpes
=======

**Status**:

  application definition, extends :ref:`NXobject`

**Description**:

  This is an application definition for angular resolved photo electron spectroscopy.

  It has been drawn up with hemispherical electron analysers in mind.

**Symbols**:

  No symbol table
